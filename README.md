# Basic Canteen Management System

A PHP-based food ordering system for educational institutions.

## Features
- User registration/login
- Shop browsing
- Menu management
- Shopping cart
- Order history
- Admin dashboard
- Customer management

## Requirements
- PHP 7.4+
- MySQL 5.7+
- Web server (Apache/Nginx)

## Installation
1. Clone repository to web root
2. Import `database/basiccanteen.sql`
3. Configure database credentials in `core/conn_db.php`
4. Set proper file permissions
5. Access via web browser
